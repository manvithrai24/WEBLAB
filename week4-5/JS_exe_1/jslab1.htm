Java Script lab Exercises-1

Q1. Define a function max() that takes two numbers as arguments from user and returns the largest of them. Use the if-then-else construct available in Javascript.

Q2. Define a function maxOfThree() that takes three numbers as arguments and returns the largest of them.

Q3. Write a function isVowel() that takes a character (i.e. a string of length 1) and returns true if it is a vowel, false otherwise.

Q4. Write a function translate() that will translate a text into "rövarspråket". That is, double every consonant and place an occurrence of "o" in between. For example, translate("this is fun") should return the string "tothohisos isos fofunon".

Q5. Define a function sum() and a function multiply() that sums and multiplies (respectively) all the numbers in an array of numbers. For example, sum([1,2,3,4]) should return 10, and multiply([1,2,3,4]) should return 24.

Q6. Define a function reverse() that computes the reversal of a string. For example, reverse("jag testar") should return the string "ratset gaj".

Q7. Represent a small bilingual lexicon as a Javascript object in the following fashion {"merry":"god", "christmas":"jul", "and":"och", "happy":gott", "new":"nytt", "year":"år"} and use it to translate your Christmas cards from English into Swedish.

Q8. Write a function findLongestWord() that takes an array of words and returns the length of the longest one.

Q9. Write a function filterLongWords() that takes an array of words and an integer i and returns the array of words that are longer than i.

Q10. Write a function charFreq() that takes a string and builds a frequency listing of the characters contained in it. Represent the frequency listing as a Javascript object. Try it with something like charFreq("abbabcbdbabdbdbabababcbcbab").

Q11. Design a calculator using java script.

Q12. Write a script that asks the user for a series of names until the user hits the cancel button. The script then sorts the names in alphabetical order and prints the names in an ordered list.

Q13. Write a script that asks the user how many random numbers they wish to generate. The script then generates the numbers and calculates the average of the numbers and returns the results to the user
